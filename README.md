🎬 VK Бот для поиска фильмов
Асинхронный бот для VK, который ищет фильмы и сериалы через API ПоискКино.

✨ Возможности
🔍 Поиск по названию — найди фильм или сериал по его названию

⭐ Поиск по рейтингу — покажи фильмы с рейтингом от N до 10

💰 Поиск по бюджету:

Низкий бюджет: от 0 до N

Высокий бюджет: от N до ∞

📋 Пагинация — листай результаты вперёд и назад

🖼️ Постеры — ссылки на обложки фильмов

📅 История запросов — просмотр всех твоих поисков

📆 История за дату — фильтрация истории по дате

✅ Отметка просмотра — отмечай просмотренные фильмы

⭐ Оценка фильма — ставь оценку от 1 до 10

🎭 Список жанров — посмотри все доступные жанры

🛠️ Технологии
Python 3.14 — основной язык

FastVK — фреймворк для VK ботов

SQLAlchemy 2.0 — ORM для работы с БД

PostgreSQL — база данных

Alembic — миграции

httpx — асинхронные запросы к API

📦 Установка и запуск
1. Клонируй репозиторий
bash
git clone https://github.com/Vanusha61/vk_bot_films_api.git
cd vk_bot_films_api
2. Создай виртуальное окружение
bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
3. Установи зависимости
bash
pip install -r requirements.txt
4. Настрой переменные окружения
Создай файл .env в корне проекта и заполни его:

env
# VK Бот
VK_API=токен_группы_ВК

# API Кинопоиска (https://poiskkino.dev/)
X_API_KEY=ключ_API_Кинопоиска

# PostgreSQL
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=твой_пароль
DB_NAME=vk_bot_db
Где взять токены:
Токен	Где взять
VK_API	Создай группу ВК, включи Long Poll API, возьми токен в настройках группы
X_API_KEY	Зарегистрируйся на poiskkino.dev и получи API ключ
PostgreSQL	Установи локально или используй Docker
5. Примени миграции
bash
alembic upgrade head
6. Запусти бота
bash
python main.py
📋 Команды бота
Команда	Описание
Список жанров	Показать все доступные жанры
Поиск по названию	Найти фильм по названию
По рейтингу	Найти фильмы с рейтингом от 1 до 10
Низкий бюджет	Найти фильмы с бюджетом до указанной суммы
Высокий бюджет	Найти фильмы с бюджетом от указанной суммы
История	Показать историю всех запросов
История за дату	Показать историю за конкретную дату
🗄️ Структура базы данных
Таблица history
Поле	Тип	Описание
id	Integer	Первичный ключ
user_vk_id	Integer	VK ID пользователя
user_name	String	Имя пользователя
command	String	Название команды
query	String	Поисковый запрос
result_preview	Text	Краткий результат
next_page_url	String	Ссылка на следующую страницу
prev_page_url	String	Ссылка на предыдущую страницу
is_watched	Boolean	Просмотрен/не просмотрен
evaluation	Integer	Оценка пользователя (1-10)
created_at	DateTime	Дата и время запроса
