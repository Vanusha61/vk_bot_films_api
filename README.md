# 🎬 VK Бот для поиска фильмов

Асинхронный бот для VK, который ищет фильмы и сериалы через API ПоискКино.

## ✨ Возможности

- 🔍 **Поиск по названию** — найди фильм по его названию.
- ⭐ **Поиск по рейтингу** — покажи фильмы с рейтингом от N до 10.
- 💰 **Поиск по бюджету**:
  - Низкий бюджет: от 0 до N.
  - Высокий бюджет: от N до ∞.
- 📋 **Пагинация** — листай результаты вперёд и назад.
- 🖼️ **Постеры** — ссылки на обложки фильмов.
